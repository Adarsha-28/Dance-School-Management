# Dance Academy Management System

## Overview

Dance Academy Management System is a web-based application developed to simplify the management of dance academy activities. The system provides an easy-to-use platform for students, parents, instructors, and administrators to interact with the academy digitally.

The application allows users to explore dance courses, submit enquiries, apply for admission, create accounts, manage student information, and access academy-related resources through a modern and responsive user interface.

## Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/dance-academy-management.git
```

### Navigate to Project Folder

```bash
cd dance-academy-management
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm start
```

The application will run at:

```text
http://localhost:3000
```
